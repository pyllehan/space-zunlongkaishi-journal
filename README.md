# space-zunlongkaishi-journal

## 项目简介

这是一个用于归档和发布多个独立 HTML 页面的仓库。  
本仓库不针对任何特定域名或网站，仅作为页面存储与历史记录之用。  
所有 HTML 文件均为独立页面，彼此不构成关联站点。

## 目录说明

```
.
├── README.md          # 本文件
├── index.html         # 仓库首页/导航页（如存在）
├── pages/             # 存放归档的 HTML 页面
│   ├── example1.html
│   ├── example2.html
│   └── ...
└── assets/            # 页面引用的资源文件（图片、样式、脚本等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/` 目录用于存放主要的 HTML 归档页面。
- `assets/` 目录用于存放页面所需的静态资源，按类型分类。

## 页面归档说明

- 本仓库中的 HTML 页面均为独立归档，不保证页面之间的链接或跳转有效性。
- 每个页面可能包含不同的主题、样式或功能，彼此无统一模板。
- 归档页面可能包含历史版本、实验性内容或临时记录，不视为正式发布产物。
- 如页面引用外部资源，请自行确认其可用性与安全性。
